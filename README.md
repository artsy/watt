# Watt

WAT is Watt?  Watt is a shared js/css/img asset library for Artsy Partner Engineering Rails apps.

## Installation

Add this line to your application's Gemfile:

    gem 'watt'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install watt
